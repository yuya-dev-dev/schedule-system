## 4. 実機Safari確認

| 項目 | 状態 | 理由・実施条件 |
| --- | --- | --- |
| 個人iPhone Safari | 保留 | PCにiPhoneから到達できるIPv4ネットワーク接続がなく、実機接続条件が成立しなかった |
| 社用スマートフォン | フェーズ6 | 会社が許可した架空データ限定の共有URLが必要 |

個人iPhone Safariでは、同じネットワークから共有URLを開ける環境を用意した後、次を確認する。

1. 現在月で今日または次の勤務日列が見える
2. 縦横スクロールとピンチ拡大ができる
3. 先頭セルと矢印セルから同じ詳細を開ける
4. 詳細を閲覧して一覧へ戻れる
5. 前月・当月・翌月タブを操作できる
6. sticky見出し、固定月タブ、Safariのアドレスバーが重ならない

## 5. 判定

- PC主要表示とiPhone相当幅の閲覧要件は合格
- 今回の範囲でアプリの重大不具合は検出されなかった
- フェーズ4完了には個人iPhone Safariの実機確認が残る
- 正式導入判断には、フェーズ6で共有URL、社用端末、共有DB再起動を追加確認する
